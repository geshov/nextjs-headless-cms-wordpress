## Пример подключения к Headless CMS (Wordpress) из приложения Next.js посредством REST API

В примере используется Next.js 16 (App Router) с активацией Cache Components.

Поиск и сортировка выполняются средствами CMS (SQL), а приложение Next.js только отображает уже отфильтрованный и отсортированный результат. Изображения тоже подгружаются из CMS.

Рабочая версия этого примера по [ссылке](https://front.geshov.ru/).
